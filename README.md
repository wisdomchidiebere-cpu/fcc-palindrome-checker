# FCC Palindrome Checker

Working through the FreeCodeCamp JavaScript Algorithms certification. This is my code for the palindrome checker project. 

The main focus here is string cleaning. It uses a regex pattern to strip out all non-alphanumeric characters, spaces, and punctuation, standardizes the casing, and then runs the comparison. 

### To test:
Run `node data_cleaner.js` in your terminal.
